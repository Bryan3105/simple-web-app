# Simple Web Application

A basic web application built with Python, Flask, PostgreSQL, Docker, and Kubernetes.

## Tech Stack

*   Frontend: HTML, CSS, JavaScript
*   Backend: Python, Flask
*   Database: PostgreSQL
*   Containerization: Docker
*   Orchestration: Kubernetes
